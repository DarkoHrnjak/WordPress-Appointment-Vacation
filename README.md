# 📅 Modern Appointment Booking System

Welcome to the **Appointment Booking Plugin**, a premium, high-end booking solution built for WordPress. Unlike standard plugins, this system focuses on an **"Apple-inspired"** modern tech aesthetic with glassmorphic elements and absolute precision.

---

## ✨ Key Features

### 💎 Premium User Experience
*   **Glassmorphism UI**: A stunning, translucent frontend form with backdrop-blur effects and vibrant indigo-to-purple gradients.
*   **Micro-Animations**: Shimmering button hover effects, smooth entry transitions, and interactive validation shaking.
*   **Real-time Validation**: Strict time boundaries (9:00 AM – 4:00 PM) and phone number sanitization that blocks letters instantly.

### ⚙️ Powerful Admin Panel
*   **Card-based Layout**: Replaced dry WordPress tables with elevated modern cards and hover-scale row highlights.
*   **Vacation Management**: Block entire date ranges with one click to prevent accidental bookings.
*   **Search & Filters**: Quickly locate appointments by date, name, or phone number.

### 🛡️ Built-in Security
*   **PHP Regex Validation**: Multi-layer security ensures only valid phone numbers and future dates enter your database.
*   **Cache Busting**: Version-controlled assets to ensure all users see your latest design immediately.

---

## 🚀 Getting Started

### 1. Installation
This project comes pre-installed as a WordPress plugin. Simply activate it in your plugin list (`v2.0.0+`).

### 2. Usage
To display the premium booking form on any page or post, simply paste this shortcode:
`[appointment_booking_form]`

### 3. Management
Head to **Appointment Booking** in your WordPress sidebar to:
*   Manage existing appointments.
*   View your schedule.
*   Set your vacation dates to block off the calendar.

## 🛠️ Tech Stack
*   **Frontend**: HTML5, Vanilla CSS (Glassmorphism), jQuery (AJAX).
*   **Backend**: PHP (WordPress Plugin API), MySQL.
*   **Design System**: Inter / System-Sans font stack, Indigo-Purple gradient brand (`#6366f1` to `#a855f7`).
